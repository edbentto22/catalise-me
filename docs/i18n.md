# Tradução estática

O site mantém o português como origem e gera versões estáticas em inglês (`/en`) e espanhol latino-americano (`/es`). O fluxo é inspirado no Verbosia: só segmentos novos ou alterados chamam o modelo; a memória é versionada em `.verbosia/page-translations.json`.

## Gerar ou atualizar traduções

1. Configure `OPENAI_API_KEY` no seu terminal ou nas variáveis do Coolify. A chave é usada apenas durante a geração e nunca chega ao browser.
2. Execute `npm run i18n:status` para conferir quantos segmentos precisam de tradução.
3. Execute `npm run i18n:translate`.
4. Revise os arquivos em `src/pages/en/` e `src/pages/es/`, e a memória em `.verbosia/`.
5. Faça o build normal: `npm run build`.

O gerador não roda automaticamente no build. Isso preserva deploys previsíveis e evita consumo inesperado de API. Depois da primeira geração, as páginas localizadas ativam o seletor de idioma, `hreflang`, `x-default`, canonical e `og:locale`.
